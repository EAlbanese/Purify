CSS Hover Animation

Code

index.html

<h1>Hover Me<h1>

stylesheet.css

								
// die zwei unteren Klassen definieren, was passiert, wenn man mit dem Zeiger über
den Text und was passiert, wenn man weiter/weg fährt.
// D.h. sobald der Zeiger auf dem Text ist kommt der blaue Balken
h1::before {  
	transform: scaleX(0);
	transform-origin: bottom right;
}
// beim wegfahren, geht er wieder zurück
h1:hover::before {
	transform: scaleX(1);
	transform-origin: bottom left;
}

// hier wird der Style für den Hover bestimmt
h1::before {
	content: " ";
	display: block;
	position: absolute;
	top: 0; right: 0; bottom: 0; left: 0;
	inset: 0 0 0 0;
	background: hsl(200 100% 80%);
	z-index: -1;
	transition: transform .3s ease;
}

// Hier wir der allgemeine Style definiert
h1 {
	position: relative;
	font-size: 4rem;
}

html {
	block-size: 100%;
	inline-size: 100%;
}

body {
	min-block-size: 100%;
	min-inline-size: 100%;
	margin: 0;
	box-sizing: border-box;
	display: grid;
	place-content: left;
	font-family: system-ui, sans-serif;
}

@media (orientation: landscape) {
	body {
		grid-auto-flow: column;
	}
}

CSS Loading Animation

Code

index.html

<span class="loader"></span>

stylesheet.css


.loader {
	width: 100px;
	height: 100px;
	position: relative;
}
// der die animation des Kreises wird bestimmt
.loader::before, .loader::after {
	content: '';
	position: absolute;
	width: inherit;
	height: inherit;
	border-radius: 50%;
	mix-blend-mode: multiply;
	animation: rotate 1s infinite
	cubic-bezier(0.77, 0, 0.175, 1);
}
// Bei den unter beinen Klassen sind die Beiden Farben beschrieben
.loader::before {
	background-color: #fc3f9e;
}
.loader::after {
	background-color: #50e8f3;
	animation-delay: .5s;
}
// Mit den Keyframes wird dargestellt, wann der Kreis welche Farbe hat
@keyframes rotate {
	0%, 100%{
		left: 95%;
	}
	25%{
		transform: scale(.3);
	}
	50%{
		left: 0;
	}
	75%{
		transform: scale(1);
	}
}

Keyframe

Durch die CSS-at-Regel @keyframes kann der Autor die einzelnen Schritte einer CSS-Animationssequenz festlegen, indem er Keyframes (oder Wegpunkte) setzt, die während der Sequenz an bestimmten Punkten erreicht werden. Das ermöglicht eine spezifischere Kontrolle über die Zwischenschritte einer Animationssequenz gegenüber dem Ergebnis einem dem Browser überlassenen, automatisch berechneten Ablauf.
Hier ein Beispiel dazu:

Synthax 

@keyframes <bezeichner> {
	[ [ from | to | <Prozentzahl> ] [, from | to | <Prozentzahl> ]* block ]*
}

Bezeichner: name um den Keyframe zu initialisieren
From: min 0%
To: Endwert max 100%
Funktioniert mit den allen gängigen Browsern.

Code

index.html

<div></div>

stylesheet.css


div {
	width: 100px;
	height: 100px;
	background-color: red;
	// nimmt die Animation example und stellt die Dauer der Animation (4s) und die wie lange die Animaiton durchefürt wird (infinite)
	animation: example 4s infinite; 
}

// beschreibt die Farben des Rechteckes, zu welchem Zeitpunkt
@keyframes example {
	0%   {background-color: red;}
	25%  {background-color: yellow;}
	50%  {background-color: blue;}
	100% {background-color: green;}
}

GIF (Graphics Interchange Format)

Blackthorn.gif

Das GIF-Format wurde 1987 (GIF87a) von dem Unternehmen CompuServe entwickelt, um Rasterbilder in Netzwerken zu übertragen. 1989 wurde das Format modifiziert (GIF89a) und Transparenz sowie Animation hinzugefügt. GIF ist ein Format für den Austausch von Bildern. Es ist eine beliebte Format-Grafik. Es können komprimierte Daten ohne Qualitätsverlust gespeichert werden, wenn 256 Farben nicht überschritten werden.

Im Internet gibt es meherere Converter, um ein GIF zu erstellen. In diesem Beispiel habe ich diesen genommen. Damit habe ich eine mp4 Datie in ein gif konvertiert.

Canvas

Das HTML-Element <canvas> wird verwendet, um Grafiken auf einer Webseite zu zeichnen. Die Grafik oben wurde mit <canvas> erstellt. Sie zeigt zwei Elemente: ein Strich und ein Rechteck mit Farbverlauf

Strich

Your browser does not support the HTML canvas tag.
Code

index.html

<canvas id="mycanavas" width="200" height="100" style="border:1px solid #d3d3d3;">Your browser does not support the HTML canvas tag.</canvas>

javascript.js


var c = document.getElementById("mycanavas");
var ctx = c.getContext("2d");
ctx.moveTo(0,0);
ctx.lineTo(200,100);
ctx.stroke();

Farbverlauf

Your browser does not support the HTML canvas tag.
Code

index.html

<canvas id="mycanavas" width="200" height="100" style="border:1px solid #d3d3d3;">Your browser does not support the HTML canvas tag.</canvas>

javascript.js


var c = document.getElementById("mycanavas");
var ctx = c.getContext("2d");
// Create gradient
var grd = ctx.createLinearGradient(0,0,200,0);
grd.addColorStop(0,"red");
grd.addColorStop(1,"white");
// Fill with gradient
ctx.fillStyle = grd;
ctx.fillRect(10,10,150,80);

Scroll Animation (CSS und javascript)

Code

index.html


<svg width="100" height="100" viewBox="0 0 24 24">
	<path d="M21,9H15V22H13V16H11V22H9V9H3V7H21M12,2A2,2 0 0,1 14,4A2,2 0 0,1 12,6C10.89,6 10,5.1 10,4C10,2.89 10.89,2 12,2Z" />
</svg>

stylesheet.css


svg {
	position: fixed;
	top: 50%;
	left: 50%;
	margin-top: -50px;
	margin-left: -50px;
	
	animation: rotate 1s linear infinite;
	animation-play-state: paused;
	animation-delay: calc(var(--scroll) * -1s);
	
	animation-iteration-count: 1;
	animation-fill-mode: both;
}
	
@keyframes rotate {
	to {
		transform: rotate(360deg);
	}
}
	
body {
	min-height: 500vh;
}

javascript.js


window.addEventListener(
  "scroll",
  () => {
    document.body.style.setProperty(
      "--scroll",
      window.pageYOffset / (document.body.offsetHeight - window.innerHeight)
    );
  },
  false
);

SVG

SVG ist ein Bilddateiformat, das speziell für die Gestaltung zweidimensionaler Vektor- und Vektor-Raster-Grafiken für Websites entwickelt wurde. SVG unterstützt Animationen, Transparenz, Farbverläufe und ist ohne Qualitätsverlust leicht skalierbar.

<img>-Tag

SVG kann auf mehrere Arten in HTML dargestellt werden. Eine davon ist mittels <img>-Tag

Code 

<img src="images/child.svg" alt="SVG mit img laden" width="10%" >
<svg>-Tag

Eine Andere Art wäre mittels <svg>-Tag

Code 

<svg width="100" height="100" viewBox="0 0 24 24">
	<path d="M21,9H15V22H13V16H11V22H9V9H3V7H21M12,2A2,2 0 0,1 14,4A2,2 0 0,1 12,6C10.89,6 10,5.1 10,4C10,2.89 10.89,2 12,2Z" />
</svg>
